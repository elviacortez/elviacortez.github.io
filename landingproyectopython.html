Visión general del proyecto

Para este proyecto, asumirá el papel de un científico de datos / analista de datos que trabaja para una nueva empresa de inversión que ayuda a los clientes a invertir su dinero en acciones. Su trabajo consistirá en extraer datos financieros como el precio histórico de las acciones y los informes trimestrales de ingresos de diversas fuentes utilizando bibliotecas Python y webscraping sobre acciones populares. Tras recopilar estos datos, los visualizarás en un cuadro de mandos para identificar patrones o tendencias. Las acciones con las que trabajaremos son Tesla, Amazon, AMD y GameStop.

Pasos

  • Extracción de datos bursátiles mediante una biblioteca de Python
  • Extracción de datos bursátiles mediante Web Scraping
  • Crear cuenta en Watson Estudio
  • Añadir cuaderno a Watson Estudio
  • Compartir cuaderno desde Watson Estudio
  • Análisis de los datos históricos de existencias/ingresos y creación de un cuadro de mando

Certificado

Qué aprendí:
- Desempeñé el papel de un científico de datos / analista de datos trabajando en un proyecto real.
- Demostré mis habilidades en Python, el lenguaje preferido para la Ciencia de Datos y el Análisis de Datos.
- Apliqué los fundamentos de Python, las estructuras de datos de Python y el trabajo con datos en Python.
- Construí un cuadro de mando utilizando Python y bibliotecas como Pandas, Beautiful Soup y Plotly utilizando el cuaderno Jupyter.
